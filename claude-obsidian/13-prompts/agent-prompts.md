# Prompt-Bausteine

## Beschreibung
Wiederverwendbare Prompt-Strukturen für sichere Änderungen im Repo.

## Zweck
Gleicher Qualitätsstandard über Claude/Codex/Cursor/Windsurf.

## Relevante Dateien im Repo
- `AGENTS.md`
- `claude-obsidian/00-start/CRITICAL-RULES.md`

Task → passenden Prompt wählen → mit Brain-Links erweitern → ausführen.

## Wichtige Regeln
- Immer auf [[00-start/CRITICAL-RULES]] referenzieren.
- Immer „nur betroffene Dateien“ und „keine Breaking Changes“ setzen.

## Risiken
- Unspezifische Prompts erhöhen Risiko für ungewollte Seiteneffekte.
